# Coffee Kura Product Specification

## Core concept

中心は豆カード。そこに自宅で淹れた一杯の記録が紐づく。カフェで飲んだ一杯は別枠で記録し、将来の好み分析に利用する。

## Main navigation

- Beans
- Cafe
- Insights
- Settingsは歯車アイコンから開く

## Rating axes

- 酸味
- 苦味
- 甘味
- コク
- 香り
- 総合評価
- リピート判定

## Storage

すべてのユーザーデータと写真は、利用端末のブラウザ内に保存する。MVPではIndexedDBを使用する。
