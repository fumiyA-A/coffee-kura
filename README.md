# Coffee Kura

Coffee Kuraは、豆カードを中心に、自宅で淹れた一杯とカフェで飲んだ一杯を記録し、自分のコーヒーの好みを知るためのモバイル向けWebアプリです。

## 主な機能

- 豆カードの追加・編集・削除
- 豆に紐づくBrew記録とレシピ比較
- 過去Brewの複製とレシピテンプレート
- 抽出方法の追加・名前変更・並べ替え
- Brew Ratioの自動計算
- カフェで飲んだ一杯の記録
- 写真の複数保存
- 酸味・苦味・甘味・コク・香り・総合評価
- リピート判定とInsightsによる簡易分析
- IndexedDBへの端末内保存
- JSONバックアップ・復元
- GitHub Pagesでの公開

## 更新履歴

### v1.4

Brew記録を、再現・比較できる実験ノートへ拡張しました。
過去の一杯から良かった条件を再利用し、同じ豆での変化を追えるようにしています。

- 過去Brewのレシピを複製して新しい一杯を作成
- レシピテンプレートの作成・編集・削除・フォームへの適用
- 過去Brewからテンプレートを作成
- 抽出方法の追加・名前変更・並べ替え・未使用項目の削除
- V60、Kalita Wave、フレンチプレス、AeroPressなどの初期候補を追加
- 豆量と湯量からBrew Ratioをリアルタイム計算
- 同じ豆の新しい4件を表形式で比較
- 最新・最高・平均評価と前回比、評価推移を表示
- テンプレートと抽出方法をバックアップ対象へ追加

### v1.3

入力・編集・データ保護を中心に改善しました。

- Bean / Brew / Cafeの日付をiPhone標準カレンダーで入力
- 表示は `YYYY/MM/DD`、内部保存は `YYYY-MM-DD`
- 未保存フォームから戻る際の確認
- 必須項目・負数・ブレンド産地数の入力検証
- 登録済みロースターの再利用
- 登録済みカフェ選択時の場所自動補完
- 写真の表紙選択、最大枚数警告、個別削除、圧縮
- Brew編集・削除
- Bean削除時に紐づくBrewも一括削除
- 保存完了トースト
- バックアップ最終日時、記録件数、写真枚数、概算容量
- 旧データを新形式へ自動移行
- GitHub Actionsで公開npmレジストリを明示

### v1.2

日付入力をiPhone標準のカレンダー選択へ変更しました。  
保存形式と画面表示形式を分離し、日付の入力ミスを防ぐ設計にしました。

### v1.1

複数画像、登録済みカフェの再利用、ブレンド産地の複数選択を追加しました。  
Bean / Brew / Cafeの編集機能と、旧データ・旧バックアップとの互換性も強化しました。

### v1.0

Bean → Brew → Insightsの基本フローを実装しました。  
Cafe記録、IndexedDB保存、写真、評価、バックアップ・復元までを含む最初のMVPです。
