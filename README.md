# Coffee Kura v1.1

豆カードを中心に、自宅で淹れた一杯とカフェで飲んだ一杯を記録し、自分の好みを知るためのモバイル向けWebアプリです。

## v1.1

- Bean / Brew / Cafeに複数画像を保存
- アルバム複数選択とカメラ追加
- 画像を保存前に縮小・圧縮
- 過去に登録したカフェをドロップダウンから選択
- 日付入力・表示をYYYY/MM/DDへ統一
- BrewとCafe記録を後から編集
- 産地タイプ：シングルオリジン / ブレンド / 不明
- ブレンド時に複数産地を選択
- v1.0以前の1枚画像・単一産地データも表示可能
- バックアップv1の読み込み互換

## 起動

```bash
npm install
npm run dev -- --host 0.0.0.0
```

## ビルド

```bash
npm run build
```
