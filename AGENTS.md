# AGENTS.md

## Product

Coffee Kuraは、豆カードを中心にコーヒーの好みを発見するモバイルファーストWebアプリです。

- 中心は豆カード
- BrewはBeanに紐づく
- CafeCupは別枠
- 主画面は Beans / Cafe / Insights
- Settingsは右上の歯車
- データはブラウザのIndexedDBに保存
- 配信はGitHub Pages

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Native IndexedDB
- GitHub Pages

## Rules

- サーバー、認証、Supabaseは追加しない
- モバイルファーストで実装する
- 外部依存をむやみに増やさない
- MVP外のMap、AI、課金、SNS機能は追加しない
- ダーク背景、ゴールドアクセント、高級感のあるカードUIを維持する
